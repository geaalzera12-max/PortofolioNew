Document all animations with annotations:

SCROLL ANIMATIONS — annotate each frame:
  ► Section headings: 
    Before: opacity 0, translateX(-60px)
    After: opacity 1, translateX(0)
    Easing: cubic-bezier(0.22, 1, 0.36, 1)
    Duration: 800ms
    Trigger: 15% viewport intersection
  
  ► Text body / paragraphs:
    Before: opacity 0, translateY(40px)
    After: opacity 1, translateY(0)
    Stagger: 100ms between lines
    Duration: 600ms
  
  ► Cards:
    Before: opacity 0, translateY(60px)
    After: opacity 1, translateY(0)
    Stagger: 150ms per card
    Duration: 700ms
    Easing: cubic-bezier(0.34, 1.56, 0.64, 1) [spring]
  
  ► Key terms / numbers:
    Animate: color from text/tertiary → brand/primary
    Duration: 400ms as element enters view
